## Trạng thái:

- OPEN
- RUNNING

## Mô tả:

- Mỗi phiên đấu giá có một trạng thái xác định.
- Trạng thái quyết định hành vi hệ thống.

## Luồng trạng thái:

OPEN → RUNNING

## Ý nghĩa: 

- OPEN: phiên đã tạo, chưa bắt đầu
- RUNNING: phiên đang diễn ra, cho phép đặt giá

## Mục tiêu đạt được:

- Đặt nền tảng quản lý vòng đời phiên đấu giá.
- Chuẩn bị cho việc kiểm soát logic theo trạng thái.
