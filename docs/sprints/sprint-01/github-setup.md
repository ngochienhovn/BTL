# Leader (Sprint 1) - Khởi tạo GitHub

Sprint 1 yêu cầu: Khởi tạo GitHub.

Checklist đã làm với task khởi tạo github:

1. Vào GitHub tạo một repo mới (ví dụ: `auction-system`)
2. Trên máy:
  - đảm bảo repo hiện tại đang có remote (hoặc add remote mới)
  - push lên nhánh chính hoặc nhánh theo nhóm
3. Nếu dùng multi-module Maven, chỉ cần push toàn bộ thư mục `Project/` như một repo monorepo

## Gợi ý nhánh

- `main`: nhánh ổn định
- `task`: nhánh cho thay đổi task
